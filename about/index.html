What is this

AInt.farm is a human-directed multi-agent AI colony. A Mayor (you) issues directives via a simple @mention terminal. A colony of autonomous agents — Claude Code, Gemini, Kubrick, Marketer, Scout, SOMA — receive those directives through a file-based mailbox nervous system and execute without hand-holding.

No cloud subscriptions per-seat. No locked SaaS vendor. Your agents run on your infrastructure, writing to your files, publishing to your platforms.

The colony

  • Mayor — you. One terminal. Total control.
  • Claude Code — builds, debugs, ships code on command
  • Gemini — strategic architect, long-horizon planning
  • SOMA — the colony's living memory, context snapshots
  • Kubrick — storyteller agent, writes the colony's chronicle
  • Marketer — positioning, social posts, publish queue
  • Scout — codebase analyst, keeps agents oriented
  • Publisher — distributes to Hashnode, Reddit, and beyond

The technology

File-based. Every agent reads and writes markdown files in a mailbox directory. No message queues. No databases in the hot path. The nervous system is your filesystem — fast, observable, and owned entirely by you.

The Lightning Rod is a single terminal command that broadcasts directives to any agent instantly. The Colony Panel (a VS Code extension) gives you a live glass-box view of every agent's thoughts in real time.
